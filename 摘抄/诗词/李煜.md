# 李煜

#### 相见欢·无言独上西楼

```
无言独上西楼，月如钩。寂寞梧桐深院锁清秋。
剪不断，理还乱，是离愁，别是一般滋味在心头。
```

#### 虞美人·春花秋月何时了

```
春花秋月何时了？往事知多少。小楼昨夜又东风，故国不堪回首月明中。雕栏玉砌应犹在，只是朱颜改。问君能有几多愁？恰似一江春水向东流。
```

#### 长相思·一重山

```
一重山，两重山。山远天高烟水寒，相思枫叶丹。
菊花开，菊花残。塞雁高飞人未还，一帘风月闲。
云一涡，玉一梭。澹澹衫儿薄薄罗，轻颦双黛螺。
秋风多，雨如和。帘外芭蕉三两窠，夜长人奈何。
```

#### 忆江南·多少恨

```
多少恨，昨夜梦魂中。
还似旧时游上苑，车如流水马如龙。
花月正春风。
```
