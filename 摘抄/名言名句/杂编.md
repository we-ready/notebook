# 杂编

### 周易

```
天行健，君子以自强不息。
```

### 老子

```
胜人者有力，自胜者强。
```

### 增广贤文

```
未曾清贫难成人，不经打击老天真。
自古英雄出炼狱，从来富贵入凡尘。
醉生梦死谁成器，拓马长枪定乾坤。
挥军千里山河在，立名扬威传后人。
```

### 其他

```
少年不知勤学苦，老来方知读书迟。
```

```
一日读书一日功，一日不读十日空。
```

```
学习不怕根底浅，只要迈步总不迟。
```

