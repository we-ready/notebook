# 杂编

```
一二三四五，金木水火土；
天地分上下，日月照今古。
```

```
站如松，坐如钟；
行如风，卧如弓。
```

```
春回大地，万物复苏；
柳绿花红，莺歌燕舞；
冰雪融化，泉水叮咚；
百花齐放，百家争鸣。
```


### 其他

#### 琴棋书画

```
有山皆图画，无水不文章
```

#### 时间

```
一年之计在于春，一日之计在于晨。
```

```
一寸光阴一寸金，寸金难买寸光阴。
```

#### 用功

```
少年不知勤学苦，老来方知读书迟。
```

```
一日读书一日功，一日不读十日空。
```

```
学习不怕根底浅，只要迈步总不迟。
```

```
书山有路勤为径，学海无涯苦作舟。
```

```

```

```

```

```

```

```

```

```

```

