# 心的归回（野地百合）

```
伊甸的美景早已一去不回，受伤的白鸽难以展翅高飞，
园中的鲜花早已凋谢枯萎，哭泣的心灵何时能以安慰。

髑髅的山顶无辜羔羊替罪，永恒的救赎成就奇妙宏伟，
黑暗的大地再见黎明光辉，疲惫的心灵终又重蒙恩惠。

将你的心，归回，让你的灵高飞，
耶稣爱你，他在等你，他是你生命中的唯一宝贵，

将你的心，归回，让你的灵高飞，
耶稣爱你，他在等你，他是你生命中的永远依归。
```
